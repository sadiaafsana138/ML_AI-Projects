# 🤖 Gemini Chatbot App (Streamlit + Google Gemini API)

A simple AI-powered chatbot built using Streamlit and Google Gemini API.

## 🌐 Live Demo

https://gemini-chatbot-using-api.streamlit.app/

## 🚀 Features

-   Ask any question
-   AI responses using Gemini
-   Simple Streamlit UI
-   Secure API key using .env

## 🧠 Tech Stack

-   Python
-   Streamlit
-   Google Generative AI
-   python-dotenv

## ⚙️ Setup

### Install dependencies

pip install -r requirements.txt

### Run app

streamlit run app.py

## 🔐 Environment

Create .env file: GOOGLE_API_KEY=your_api_key
